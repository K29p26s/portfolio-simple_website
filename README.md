# Personal Portfolio Website

This project is a simple **Personal Portfolio Website** built using **HTML and basic CSS**. It showcases my interests, projects, and contact information.

## Pages Included

* **Home Page** – Displays my interests and project details.
* **Contact Page** – Provides contact information to reach me.

## Technologies Used

* HTML5
* CSS3

## How to View the Project

1. Download or clone the repository.
2. Open the project folder.
3. Double-click the **index.html** file.
4. The website will open in your web browser.
